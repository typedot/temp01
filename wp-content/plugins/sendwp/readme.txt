Say hello to the easy solution to transactional email in WordPress.

== Description ==

When this plugin is installed, you’ll be able to sign up for the SendWP service from the SendWP menu page. Once you’re signed up, all your WordPress emails will be routed through sendwp.com. We know that sending all your email through a third-party can be scary, so please can check out our privacy policy and terms and conditions.

= Installation ==

1. Install and activate SendWP.
1. Visit the SendWP admin menu page.
1. Click on the "Connect" button.

== Frequently asked questions ==

Q. Is the SendWP service free?
A. No. SendWP is a paid service.

Q. Does this plugin work without signing up for the SendWP service?
A. No. This plugin only serves to connect you to the SendWP service, so if you don’t use the service, the plugin doesn’t have anything to do.
